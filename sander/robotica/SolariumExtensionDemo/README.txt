This directory contains two demos showing how to extend the 
Solarium application. 


VirtualObjectPlugin

This demo modifies Solarium to display a SPOT that is running the SendDataDemo with a unique icon and with a modified popup menu that 
includes a command to display the data being collected graphically
in a Solarium window. 


NewViewPlugin

This demo shows how to add a new view to Solarium, MyRobotView, and
how to extend the Emulator to work with the new view.
